# Flex Box Learning Project

## Deskripsi
Proyek sederhana untuk belajar **CSS Flexbox** mengikuti tutorial dari channel YouTube **Web Programming Unpas (WPU)**. Proyek ini berisi beberapa latihan dan contoh implementasi Flexbox untuk memahami konsep layout yang lebih fleksibel dan responsive.

## Konten Project
- **index.html & style.css** - File utama dengan contoh implementasi Flexbox
- **Latihan-FlexBox/** - Folder berisi latihan-latihan Flexbox dengan gambar pendukung
- **Responsive-Navbar/** - Contoh implementasi navbar responsive menggunakan Flexbox dan JavaScript

## Teknologi
- HTML5
- CSS3 (Flexbox)
- JavaScript (untuk interaktivitas)
